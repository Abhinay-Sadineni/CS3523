/*-----------------------------------------------------------------------------------------*/
                        Input Instructions
/*-----------------------------------------------------------------------------------------*/
1)Create a file with name input.txt ,this file should be in same directory as the source code files

2)The first line of input will contain  two values K and N ,where K  is the no of threads and N
  is dimensional value of the N x N sudoku.

3)From secod line ,the elements of the sudoku will be present ,with each row in a new line.

4)Note: the input file name should be input.txt ,with any other name the code will not be executed.

5)Sample input file:
 
 4 4
 1 3 4 2
 2 4 1 3
 4 2 3 1
 3 1 2 4
 
6) save the file 



(This is done on Manjaro Linux commands, may vary for mac-os)
/*-----------------------------------------------------------------------------------------*/
                        Compile And execution Instructions for pthread
/*-----------------------------------------------------------------------------------------*/
1)Compile commands:
    gcc  Assgn2Srcpthread-cs21btech11055.c -lpthread -lm

2)execution commands:
    ./a.out

3)output.txt will be generated please be paitent until the execution is completed
